## 思路

先写出**递归**的算法，再转成**动态规划**

暴力递归思路：
总共要爬 n 阶。
按照第一次爬几个台阶来分，只有这两种情况：
1 第一次爬 1 个台阶，之后需要爬 n-1 个台阶
2 第一次爬 2 个台阶，之后需要爬 n-2 个台阶
所以，爬 n 阶的方法数 = 这两种情况的方法数之和 = ( 爬 n-1 个台阶的方法数 ) + ( 爬 n-2 个台阶的方法数 )

代码：

```js
var climbStairs = function (n) {
  if (n === 1) {
    return 1;
  } else if (n === 2) {
    return 2;
  } else {
    return climbStairs(n - 1) + climbStairs(n - 2);
  }
};
```

转成动态规划

```js
var climbStairs = function (n) {
  if (n === 1) {
    return 1;
  } else if (n === 2) {
    return 2;
  }
  let k1 = 1;
  let k2 = 2;
  for (let i = 3; i <= n; i++) {
    [k1, k2] = [k2, k1 + k2];
  }
  return k2;
};
```
